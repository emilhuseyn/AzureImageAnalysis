# Azure Image Moderation with C#

This project demonstrates how to use Azure's Computer Vision API in C# to analyze images and check if they contain adult or racy content.
It helps you easily integrate image moderation into your applications.

---

## Why Use This Service?

If your platform allows users to upload images, it's essential to ensure that the content is safe and appropriate. 
Azure's Computer Vision API offers a way to automatically detect and flag adult or racy content in images, allowing you to maintain a safe environment for your users.
